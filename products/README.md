## How to Use

- **Wall Art**: Open SVG files in any browser, Cricut Design Space, Silhouette Studio, Adobe Illustrator, or Inkscape. Print at any size.
- **Planners**: Open HTML files in any web browser. Press Ctrl+P (or Cmd+P on Mac) and select "Save as PDF" to create a printable PDF. Ensure "Background graphics" is checked in print settings.
