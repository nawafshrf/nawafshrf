How to Print Your Social Media Planner

Quick Start (4 Easy Steps)

1

Open the File

Open any of the 4 color theme HTML files in Chrome, Firefox, Safari, or Edge.

2

Print to PDF

Press Ctrl+P (Windows) or Cmd+P (Mac). Select "Save as PDF" as the destination.

3

Check Settings

Make sure "Background graphics" is checked. Set margins to "Default" and paper to "Letter".

4

Save & Print

Save the PDF, then print on your home printer or upload to a print service.

Important: You MUST check "Background graphics" (or "Background colors and images" in some browsers) for the colors and design to appear correctly.

Which Pages to Print Multiple Copies

  • Monthly Content Calendar — Print 12 copies (one per month)
  • Weekly Post Planner — Print 52 copies (one per week) or as needed
  • Monthly Analytics Review — Print 12 copies (one per month)
  • All other pages — Print 1 copy each

Suggested Planner Setup

  1. Print the Brand Voice Guide first -- this is your foundation
  2. Set up your Content Pillars to define your themes

Troubleshooting

  • Colors not showing? Make sure "Background graphics" is checked in print settings.
  • Pages cut off? Set margins to "Default" or "None" and paper size to "Letter".
  • Blank pages? Try a different browser. Chrome usually gives the best results.
  • Text too small? Make sure your browser zoom is set to 100% before printing.
